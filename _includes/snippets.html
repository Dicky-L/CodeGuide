code snippets

code snippets规范列表，同时提供sublime插件Javascript Snippets for IMWeb支持这些snippets。

说明：

[快捷代码] 解释

snippets的内容，其中占位符语法：${序号:默认值}

  • [cd] console.dir

    {% highlight javascript %}console.dir(${1:obj});{% endhighlight %}

  • [mi] new Image

    {% highlight javascript %}new Image().src = '/${1}';{% endhighlight %}
